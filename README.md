# Spatial-Transcriptomics using Merscope Data 

This repository contains analyses of **Merscope spatial transcriptomics data**, comparing **spatial and non-spatial clustering** approaches.

## Data
- Merscope spatial transcriptomics dataset

## Clustering Approaches
- **Spatial Clustering:** Using **Banksy** to incorporate spatial information. Find Banksy paper here (https://www.nature.com/articles/s41588-024-01664-3#MOESM1) and Github herre (https://github.com/prabhakarlab/Banksy)
- **Non-Spatial Clustering:** Standard clustering without spatial context  

## Analyses
- Comparison of spatial vs non-spatial clusters  
- Differential expression between clusters  
- Visualization of spatial patterns  
- Downstream cluster characterization analyses
- Deconvolution using RCTD package. Find paper here (https://www.nature.com/articles/s41587-021-00830-w) and github here (https://github.com/dmcable/spacexr?tab=readme-ov-file)
